# simple-nostr

### Command line nostr client


Interact with the [nostr network](https://github.com/nostr-protocol/nostr).

Clone
```
git clone https://github.com/ksfi/simple-nostr.git
```

Requirements
```
pip install -r requirements.txt
```


Run

```
python3 message.py
```


Send a note with a random account

```
[msg]
```

Send a note with an account keypair

```
[msg] // [privkey] // [pubkey]
```
