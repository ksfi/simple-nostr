# simple-nostr

## Command line nostr client


Interact with the [nostr network](https://github.com/nostr-protocol/nostr).

Clone
```
git clone https://github.com/ksfi/simple-nostr.git
```

Run with

```
python3 message.py
```


Send a note with a randomly generated account

```
[msg]
```

Send a note with your account keypair

```
[msg] // [privkey] // [pubkey]
```
